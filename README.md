# Notifity to Something Devices System is "Yabusame"

Android, iOS, Web にデバイスに関わらず同じ内容をPushするシステムです。現在モック作成中。

## 作成予定の機能

* Push内容の作成
* Pushの送信
* ユーザ毎のPush先の変更
	* デバイストークンの管理
	* 2垢対応
* カレンダー機能
* カテゴリ機能
* セグメンテーション機能
* 既読管理
* APNs, GCMの管理

## モック版で作成する機能

* Push内容の作成
* ユーザ毎のPush先の変更
	* デバイストークンの管理
	* 2垢対応
* カレンダー機能
* カテゴリ機能

GCM, APNsを使える様にするにはAPIKeyが必要なのでモック版では行わないでおく
